# ⚡ Premium Full Stack Portfolio | Paz Gueta

Welcome to my personal portfolio repository. This project is a highly responsive, performant, and modern web application designed to showcase my engineering background, technical stack, and production-grade case studies.

Built from scratch with a focus on fluid typography, dynamic animations, and pristine layout structure.

---

## 🚀 Key Features

- **Premium Vibrant Theme:** Includes a seamless, interactive Dark/Light mode toggle that respects user persistence via `localStorage`.
- **Fluid & Scalable Layout:** Utilizes advanced modern CSS features (`clamp()`, fluid grids, dynamic variables) ensuring flawless rendering from mobile screens up to ultra-wide displays.
- **Architectural Project Previews:** Displays high-fidelity case studies including a Hierarchical Catalog Management System and a Vehicle Management Dashboard.
- **Interactive UX Elements:** Built-in mobile responsive navigation drawers, dynamic scroll-reveals, smooth counters, and automated copyright year updating.

---

## 🛠️ Tech Stack & Ecosystem

### Frontend Engineering
* **Core:** TypeScript, JavaScript (ES6+), HTML5
* **Frameworks & UI:** React, Angular, Tailwind CSS, Vite, Redux
* **Real-time & Tooling:** Socket.IO, Advanced Intersection Observers

### Backend & Architecture
* **Runtimes & Frameworks:** Node.js, Nest.js, Express.js, Fast API (Python)
* **API Documentation:** Swagger

### Databases & DevOps
* **Storage Systems:** PostgreSQL, MongoDB, MySQL
* **Deployment & Environments:** Docker, Amazon Web Services (AWS)

---

## 📂 Project Showcase (Included Case Studies)

1. **Hierarchical Catalog Management System**
   - Supports nested categories, role-based inheritance models, and secure permission-aware filtering.
   - *Stack:* React, TypeScript, Nest.js, MongoDB, Docker

2. **Vehicle Management & Booking System**
   - A fully functional vehicle desk platform enabling approval workflows, ride reservations, and real-time tracking via WebSockets.
   - *Stack:* Angular, Python, Fast API, PostgreSQL, Socket.IO

3. **Booking.com Clone**
   - High-fidelity replication of dynamic availability search logic and structured database hotel-room relationships.
   - *Stack:* Node.js, MySQL, TypeScript, Redux

4. **ML Apartment Recommender**
   - Machine learning-driven engine that aggregates real estate data via web scraping and suggests items based on complex user budgets and preferences.
   - *Stack:* React, Python, Web Scraping Algorithms
